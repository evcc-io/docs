# Victron VRM Solar Forecast

## Templateparameter `victron`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| VRM-Installations-ID der Installation<br/>`idsite` | Die VRM-Installations-ID wird in den Einstellungen der Installation unter "Allgemeines" angezeigt | Beispiel: `123456` | ja |
| API Zugriffstoken<br/>`token` | Token können im VRM erstellt werden unter Präferenzen->Integrationen |  | ja |
| Intervall<br/>`interval` |  | Standard: `1h` | nein |
