## Templateparameter `nordpool`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Region<br/>`region` |  | Beispiel: `GER`<br/>Auswahl: `EE`, `LT`, `LV`, `AT`, `BE`<br/>, `FR`, `GER`, `NL`, `PL`, `DK1`<br/>, `DK2`, `FI`, `NO1`, `NO2`, `NO3`<br/>, `NO4`, `NO5`, `SE1`, `SE2`, `SE3`<br/>, `SE4`, `TEL`, `SYS` | nein |
| Währung<br/>`currency` |  | Standard: `EUR`<br/>Auswahl: `DKK`, `EUR`, `NOK`, `PLN`, `RON`, `SEK` | ja |
| Aufschlag<br/>`charges` | Zusätzlicher fester Aufschlag pro kWh (z.B. 0.05 für 5 Cent) |  | nein |
| Steuer<br/>`tax` | Zusätzlicher prozentualer Aufschlag (z.B. 0.2 für 20%) |  | nein |
| Formel<br/>`formula` | Individuelle Formel zur Berechnung des Preises | Beispiel: `math.Max((price + charges) * (1 + tax), 0.0)` | nein |
