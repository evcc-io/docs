# Solcast

<div className="device-detail-container">
  <div className="device-detail-content">

Benötigt einen [solcast.com](https://solcast.com/free-rooftop-solar-forecasting)-Account. Der kostenlose "Home User" Tarif ist für private Anwendungen oft ausreichend. Dieser Plan hat ein API-Limit von 10 Anfragen pro Tag.

## Templateparameter `solcast`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Ressource ID deiner Anlage<br/>`site` |  |  | ja |
| API Token<br/>`token` |  |  | ja |
| Startzeit<br/>`from` | Startzeit der Datenabrufe, Angabe in vollen Stunden, z.B "6" |  | nein |
| Endzeit<br/>`to` | Endzeit der Datenabrufe, Angabe in vollen Stunden, z.B "20" |  | nein |
| Intervall<br/>`interval` |  | Standard: `3h` | nein |
