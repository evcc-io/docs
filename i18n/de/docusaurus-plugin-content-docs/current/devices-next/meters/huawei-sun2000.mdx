# Huawei SUN2000

Grid und Batterie erfordern den PowerSensor.
"Modbus/TCP" erfordert Freischaltung via "Errichterzugang" in den Kommunikationseinstellungen des Wechselrichters.
Siehe https://forum.huawei.com/enterprise/en/modbus-tcp-guide/thread/667250677153415168-667213868771979264

## Templateparameter `huawei-sun2000`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `rs485`, `tcpip` | nein |
| Zeitüberschreitung<br/>`timeout` |  | Standard: `15s`<br/>Beispiel: `10s` | nein |
| Nummer des Batteriespeichers<br/>`storageunit` |  | Standard: `1` | nein |
| Maximale Ladeleistung<br/>`maxchargepower` | Für erzwungenes Laden des Speichers. | Standard: `10000`<br/>Einheit: W | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Wechselrichterkaskade<br/>`forceaccharging` | AC-Laden bleibt aktiv zum Laden des Speichers aus anderen AC Quellen für Wechselrichterkaskaden. Verhindert Stand-by. |  | nein |
