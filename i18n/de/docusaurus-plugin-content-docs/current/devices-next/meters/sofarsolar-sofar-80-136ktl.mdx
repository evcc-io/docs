# SofarSolar SOFAR 80…136KTL

<div className="device-detail-container">
  <div className="device-detail-content">
    <DeviceFeatures features="battery-control" />

Die Verbindung über einen LSE-3 Logger Stick mittels LAN Anschluss und ModBus TCP über Port 8899 ist die einfachste Anbindung. Der LSW-3 WLAN Stick wird nicht unterstützt. Bei seriellem Anschluss via RS485 am COM Port ist eine wechselrichterseitige Terminierung notwendig.

## Templateparameter `sofarsolar-g3`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `tcpip`, `rs485` | nein |
| Nummer des Batteriespeichers<br/>`storageunit` | Im Fall eines BTS Speichers nicht die Adresse eines BTS 5K Batteriemodules, sondern der Speicherturm (BTS 5K-BDU Steuerungseinheit mit 1-4 BTS 5K Modulen). | Standard: `1` | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Standardmodus für die aktive Batteriesteuerung<br/>`defaultmode` | Gültige Werte sind 0 (Eigenbedarfsmodus), 1 (Nutzungszeitmodus), 2 (Zeitmodus), 4 (Peak-shaving Modus) |  | nein |
