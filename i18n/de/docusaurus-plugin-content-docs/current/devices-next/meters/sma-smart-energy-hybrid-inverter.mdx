## Templateparameter `sma-hybrid`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| `modbus` | Der ModbusTCP Server muss aktiviert sein. | Auswahl: `tcpip` | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Maximale Ladeleistung<br/>`maxchargepower` | Für erzwungenes Laden des Speichers. | Standard: `4200`<br/>Einheit: W | nein |
| `watchdog` |  | Standard: `60s` | nein |
