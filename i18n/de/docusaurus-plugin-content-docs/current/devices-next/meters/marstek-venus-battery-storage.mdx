## Templateparameter `marstek-venus`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| `modbus` |  | Auswahl: `rs485` | nein |
| Akkukapazität<br/>`capacity` | Venus-E 5.12 kWh, Venus-C 2.56 kWh | Standard: `5.12`<br/>Beispiel: `50`<br/>Einheit: kWh | nein |
| Venus kann nur bis 11% entladen werden.<br/>`minsoc` | Untere Grenze beim Entladen der Batterie im normalen Betrieb | Standard: `11`<br/>Beispiel: `25`<br/>Einheit: % | nein |
| Maximaler Ladestand<br/>`maxsoc` | Oberes Limit beim Laden der Batterie aus dem Netz | Standard: `100`<br/>Beispiel: `95`<br/>Einheit: % | nein |
| Ladeleistung in Watt für den Charge-Modus. 2500W ist das Maximum für Venus.<br/>`maxchargepower` | Für erzwungenes Laden des Speichers. | Standard: `2500`<br/>Einheit: W | nein |
| Work mode für Normal-Modus. 0 manuell, 1 Eigenverbrauch, 2 AI-Optimierung.<br/>`work_mode_normal` |  | Standard: `1` | nein |
