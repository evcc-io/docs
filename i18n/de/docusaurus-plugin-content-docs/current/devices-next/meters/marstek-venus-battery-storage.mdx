## Templateparameter `marstek-venus`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `rs485` | nein |
| Akkukapazität<br/>`capacity` | Venus-E 5.12 kWh, Venus-C 2.56 kWh | Standard: `5.12`<br/>Beispiel: `50`<br/>Einheit: kWh | nein |
| Minimaler Ladestand<br/>`minsoc` | Venus kann nur bis 11% entladen werden. | Standard: `11`<br/>Beispiel: `11`<br/>Einheit: % | nein |
| Maximaler Ladestand<br/>`maxsoc` | Oberes Limit beim Laden der Batterie aus dem Netz | Standard: `100`<br/>Beispiel: `95`<br/>Einheit: % | nein |
| Maximale Ladeleistung<br/>`maxchargepower` | 2500W ist das Maximum für Venus. | Standard: `2500`<br/>Einheit: W | nein |
| Work mode für Normal-Modus<br/>`work_mode_normal` | 0=manuell, 1=Eigenverbrauch, 2=AI-Optimierung. | Standard: `1` | nein |
