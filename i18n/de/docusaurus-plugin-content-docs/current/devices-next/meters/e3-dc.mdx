## Templateparameter `e3dc-rscp`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` |  | Beispiel: `192.0.2.2` | ja |
| Port<br/>`port` |  | Standard: `5033` | nein |
| Benutzerkonto<br/>`user` | bspw. E-Mail Adresse, User Id, etc. |  | ja |
| Passwort<br/>`password` | Bei führenden Nullen bitte in einfache Hochkommata setzen |  | ja |
| RSCP-Password<br/>`key` |  |  | ja |
| Entladelimit in W<br/>`dischargelimit` | Limitiert die Entladeleistung im 'Halten' Batteriemodus |  | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
