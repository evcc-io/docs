## Templateparameter `sma-sbs-modbus`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` | Der ModbusTCP Server muss aktiviert sein. | Auswahl: `tcpip` | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale Ladeleistung<br/>`maxchargepower` | Für erzwungenes Laden des Speichers. | Standard: `4200`<br/>Einheit: W | nein |
| Watchdog<br/>`watchdog` |  | Standard: `60s` | nein |
