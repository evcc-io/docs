## Templateparameter `openems`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` |  | Beispiel: `192.0.2.2` | ja |
| Passwort<br/>`password` | Bei führenden Nullen bitte in einfache Hochkommata setzen | Standard: `user` | nein |
| Steuerbare Batterie Komponente<br/>`battery` |  | Beispiel: `ess0` | nein |
| Watchdog<br/>`watchdog` | abgestimmt auf das API-Timeout | Standard: `60s` | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
