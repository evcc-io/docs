## Templateparameter `saj-h2`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `rs485`, `tcpip` | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Standardmodus für die aktive Batteriesteuerung<br/>`defaultmode` | Wechselrichter fällt nach einem Laden des Speichers oder Unterbinden der Entladung zurück auf diesen Modus. | Standard: `2` | nein |
| Minimaler Ladestand<br/>`minsoc` | Untere Grenze beim Entladen der Batterie im normalen Betrieb | Standard: `20`<br/>Beispiel: `25`<br/>Einheit: % | nein |
| Maximaler Ladestand<br/>`maxsoc` | Oberes Limit beim Laden der Batterie aus dem Netz | Standard: `95`<br/>Beispiel: `95`<br/>Einheit: % | nein |
