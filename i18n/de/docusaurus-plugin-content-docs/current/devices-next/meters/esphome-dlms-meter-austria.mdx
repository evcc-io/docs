# ESPHome DLMS Meter Austria

<DeviceFeatures features="" />

Benötigt ein ESPHome Gerät mit der `dlms_meter` Komponente (z.B. von `github://SimonFischer04/esphome@dlms-meter`),
konfiguriert für österreichische DLMS-Zähler.


<DeviceConfig code={`meters:
    - name: my_grid
      type: template
      template: esphome-dlms-austria
      usage: grid
      host: 192.0.2.2 # IP-Adresse oder Hostname`} advanced={`meters:
    - name: my_grid
      type: template
      template: esphome-dlms-austria
      usage: grid
      host: 192.0.2.2 # IP-Adresse oder Hostname
      timeout: 10s # Zeitüberschreitung, optional`} />

## Templateparameter `esphome-dlms-austria`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` |  | Beispiel: `192.0.2.2` | ja |
| Zeitüberschreitung<br/>`timeout` |  | Standard: `10s`<br/>Beispiel: `10s` | nein |
