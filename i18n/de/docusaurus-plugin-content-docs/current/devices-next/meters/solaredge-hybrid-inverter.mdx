# SolarEdge Hybrid Inverter

<div className="device-detail-container">
  <div className="device-detail-content">
    <DeviceFeatures features="battery-control" />

Nur ein System kann und darf zeitgleich eine Modbus TCP-Verbindung zum Wechselrichter haben!
Für die optionale Batteriesteuerung muss StorageConf_CtrlMode (0xE004) auf 4 "Remote" stehen.

## Templateparameter `solaredge-hybrid`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `tcpip`, `rs485` | nein |
| Zeitüberschreitung<br/>`timeout` |  | Beispiel: `10s` | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Watchdog<br/>`watchdog` |  | Standard: `60s` | nein |
