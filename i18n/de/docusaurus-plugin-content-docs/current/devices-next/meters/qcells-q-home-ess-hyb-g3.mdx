## Templateparameter `solax`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `rs485`, `tcpip` | nein |
| Dritter PV-Eingang<br/>`mppt3` | Der Wechselrichter hat einen dritten PV-Eingang (MPPT3) |  | nein |
| Nummer des Batteriespeichers<br/>`storageunit` |  | Standard: `1` | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Standardmodus für die aktive Batteriesteuerung<br/>`defaultmode` | Wechselrichter fällt nach einem Laden des Speichers oder Unterbinden der Entladung zurück auf diesen Modus. |  | nein |
