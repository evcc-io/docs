## Templateparameter `fritzdect`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| URI<br/>`uri` | HTTP(S) Adresse | Standard: `https://fritz.box` | nein |
| Benutzerkonto<br/>`user` | bspw. E-Mail Adresse, User Id, etc. |  | ja |
| Passwort<br/>`password` | Bei führenden Nullen bitte in einfache Hochkommata setzen |  | ja |
| Aktoridentifikationsnummer (AIN)<br/>`ain` | Ist auf dem Typenschild auf der Geräterückseite aufgedruckt. Bei führenden Nullen bitte in doppelte Hochkommata setzen. | Beispiel: `307788992233` | ja |
