## Templateparameter `victron-energy`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` |  | Beispiel: `192.0.2.2` | ja |
| Port<br/>`port` |  | Standard: `502` | nein |
| `meterid` | Grid-Energiezähler VRM instance- kann im VRM Portal oder im RemoteUI ausgelesen werden. |  | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Minimaler Ladestand<br/>`minsoc` | Untere Grenze beim Entladen der Batterie im normalen Betrieb | Beispiel: `25`<br/>Einheit: % | nein |
| Maximaler Ladestand<br/>`maxsoc` | Oberes Limit beim Laden der Batterie aus dem Netz | Beispiel: `95`<br/>Einheit: % | nein |
