## Templateparameter `rct-power`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` |  | Beispiel: `192.0.2.2` | ja |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Minimaler Ladestand<br/>`minsoc` | Untere Grenze beim Entladen der Batterie im normalen Betrieb | Standard: `7`<br/>Beispiel: `25`<br/>Einheit: % | nein |
| Maximaler Ladestand<br/>`maxsoc` | Oberes Limit beim Laden der Batterie aus dem Netz | Standard: `97`<br/>Beispiel: `95`<br/>Einheit: % | nein |
| Cache<br/>`cache` | Zeitintervall nach dem Daten erneut vom Fahrzeug abgefragt werden | Standard: `30s`<br/>Beispiel: `5m` | nein |
| Maximale Ladeleistung<br/>`maxchargepower` | Für erzwungenes Laden des Speichers. | Einheit: W | nein |
| Externe Leistung<br/>`externalpower` | Externe Leistung aller an S0 angeschlossenen Geräte abfragen |  | nein |
