## Templateparameter `enphase`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` |  | Beispiel: `192.0.2.2` | ja |
| `token` | Ab Envoy Firmware D7.x.xxx notwendig. Token ist ein Jahr gültig. Anleitung (Obtaining a token via web UI): https://enphase.com/download/accessing-iq-gateway-local-apis-or-local-ui-token-based-authentication |  | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Cache<br/>`cache` | Zeitintervall nach dem Daten erneut vom Fahrzeug abgefragt werden | Standard: `1s`<br/>Beispiel: `5m` | nein |
