## Templateparameter `sungrow-hybrid`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `rs485`, `tcpip` | nein |
| Zeitüberschreitung<br/>`timeout` |  | Beispiel: `10s` | nein |
| Akkukapazität<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | nein |
| Maximale AC Leistung des Hybrid-Wechselrichters<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | nein |
| Maximale Ladeleistung<br/>`maxchargepower` | Maximale Ladeleistung für erzwungenes Laden des Speichers in Watt. (0 um automatisch den Wert der maximalen BDC-Nennleistung einzustellen) | Einheit: W | nein |
