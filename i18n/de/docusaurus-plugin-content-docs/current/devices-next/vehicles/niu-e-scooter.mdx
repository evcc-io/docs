## Templateparameter `niu-e-scooter`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Titel<br/>`title` | Wird in der Benutzeroberfläche angezeigt |  | nein |
| Icon<br/>`icon` | Wird in der Benutzeroberfläche angezeigt | Standard: `scooter`<br/>Auswahl: `car`, `bike`, `bus`, `moped`, `motorcycle`<br/>, `rocket`, `scooter`, `taxi`, `tractor`, `rickshaw`<br/>, `shuttle`, `van`, `airpurifier`, `battery`, `bulb`<br/>, `climate`, `coffeemaker`, `compute`, `cooking`, `cooler`<br/>, `desktop`, `device`, `dishwasher`, `dryer`, `floorlamp`<br/>, `generic`, `heater`, `heatexchange`, `heatpump`, `kettle`<br/>, `laundry`, `laundry2`, `machine`, `meter`, `microwave`<br/>, `pump`, `smartconsumer`, `tool`, `waterheater` | nein |
| Benutzerkonto<br/>`user` | bspw. E-Mail Adresse, User Id, etc. |  | ja |
| Passwort<br/>`password` | Bei führenden Nullen bitte in einfache Hochkommata setzen |  | ja |
| Scooter Seriennummer, wie in der NIU app angegeben<br/>`serial` |  |  | ja |
| Akkukapazität<br/>`capacity` |  | Standard: `4`<br/>Beispiel: `50`<br/>Einheit: kWh | nein |
