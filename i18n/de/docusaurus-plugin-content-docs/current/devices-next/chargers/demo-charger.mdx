      power: 0 # Leistung, optional
      enabled: true # Ladebereit, optional
      maxcurrent: 16 # Maximale Stromstärke (A), Definiert die maximale Stromstärke pro angeschlossener Phase mit welcher das Fahrzeug geladen werden soll (optional)
      phases1p3p: false # Phasenumschaltung, optional`} />

## Templateparameter `demo-charger`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Ladezustand<br/>`status` |  | Standard: `A`<br/>Auswahl: `A`, `B`, `C` | ja |
| Leistung<br/>`power` |  |  | nein |
| Ladebereit<br/>`enabled` |  | Standard: `true` | nein |
| Maximale Stromstärke<br/>`maxcurrent` | Definiert die maximale Stromstärke pro angeschlossener Phase mit welcher das Fahrzeug geladen werden soll | Beispiel: `16`<br/>Einheit: A | nein |
| Phasenumschaltung<br/>`phases1p3p` |  |  | nein |
