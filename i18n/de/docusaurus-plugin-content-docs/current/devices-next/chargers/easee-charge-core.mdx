## Templateparameter `easee`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Benutzerkonto<br/>`user` | Emailadresse |  | ja |
| Passwort<br/>`password` | wie Login für Easee App oder Web Portal (https://easee.cloud) |  | ja |
| Charger Seriennummer<br/>`charger` |  | Beispiel: `EH______` | ja |
| Zeitüberschreitung<br/>`timeout` | Spezifisches Timeout für Easee API Interaktionen. Kann Warnungen und Fehler bei träger Easee API reduzieren. | Standard: `20s`<br/>Beispiel: `10s` | nein |
| Authentifizierung aktiviert<br/>`authorize` | Steuert ob evcc die Authentifizierung am Charger vornimmt. Vorteil ist ein kontrollierter Ladestart. Nicht kompatibel mit RFID Identifikation von Fahrzeugen. |  | nein |
