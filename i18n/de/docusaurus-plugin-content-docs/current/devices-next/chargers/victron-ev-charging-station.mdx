# Victron EV Charging Station

<DeviceFeatures features="" />

Trage den Host der Wallbox (nicht des GX-Geräts) ein und stelle sicher, dass die Wallbox sich im Modus "Manual" befindet.


<DeviceConfig code={`chargers:
    - name: my_charger
      type: template
      template: victron-evcs
      
      # Modbus TCP
      modbus: tcpip
      id: 1
      host: 192.0.2.2 # Hostname
      port: 502 # Port`}  />

## Templateparameter `victron-evcs`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| `modbus` |  | Auswahl: `tcpip` | nein |
