# KEBA KeContact P40 Pro

<DeviceFeatures features="1p3p,mA,rfid,sponsorship" />

Erfordert Firmwareversion 3.10.42 (C-series) bzw. 1.11 (X-series). Zur Phasenumschaltung wird zusätzlich der Keba Phasenumschalter (KeContact S10) benötigt und in den Wallboxeinstellungen muss die Umschaltsteuerung per Modbus aktiviert werden. Bei der x-Serie im WebMenü, bei der C-Serie per Modbus durch setzen des Wertes "3" im Register 5050.

## Templateparameter `keba-modbus`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| `modbus` |  | Auswahl: `tcpip` | nein |
| Laden bei Verbindung<br/>`welcomecharge` | Wallbox gibt kurzzeitige Ladefreigabe bei Fahrzeugverbindung. Das ermöglicht es Fahrzeugen, die eine Stromversorgung beim Anschließen benötigen, einen Fehlerzustand zu vermeiden. |  | nein |
