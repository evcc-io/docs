## Templateparameter `openwb-2.0`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Modbus Typ<br/>`modbus` |  | Auswahl: `tcpip` | nein |
| Ladepunkt (falls >1 Ladepunkt)<br/>`connector` |  | Standard: `1` | nein |
| Phasenumschaltung vorhanden<br/>`phases1p3p` |  | Standard: `true` | nein |
