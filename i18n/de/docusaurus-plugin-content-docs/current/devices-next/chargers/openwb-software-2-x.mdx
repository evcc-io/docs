# openWB Software 2.x

<DeviceFeatures features="1p3p,mA,rfid" />

Erfordert [`Software 2.x`](https://github.com/openWB/core).
Folgende Änderungen sind unter dem `Einstellungen`-Reiter erforderlich:

* Steuerungsmodus: `secondary`
* Steuerung über Modbus als secondary: `An`

## Templateparameter `openwb-2.0`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| `modbus` |  | Auswahl: `tcpip` | nein |
| Ladepunkt (falls &gt;1 Ladepunkt)<br/>`connector` |  | Standard: `1` | nein |
| Phasenumschaltung vorhanden<br/>`phases1p3p` |  | Standard: `true` | nein |
