## Templateparameter `tinkerforge-warp`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` | IP Adresse oder der Hostname des MQTT Brokers | Beispiel: `192.0.2.2` | ja |
| Port<br/>`port` | MQTT Broker Port | Standard: `1883` | nein |
| Topic<br/>`topic` | Topic (ohne / am Anfang) | Standard: `warp` | nein |
| Zeitüberschreitung<br/>`timeout` | Akzeptiere keine Daten die älter sind als dieser Wert | Standard: `30s`<br/>Beispiel: `10s` | nein |
| Energiemanager MQTT Topic<br/>`energymanager` | WEM Firmware v2 erforderlich. |  | nein |
| Benutzerkonto<br/>`user` | bspw. E-Mail Adresse, User Id, etc. |  | nein |
| Passwort<br/>`password` | Bei führenden Nullen bitte in einfache Hochkommata setzen |  | nein |
