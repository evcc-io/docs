# Tessie

  <div className="device-detail-content">

Ladegerät, das über die Tessie-API verbunden ist. Ermöglicht die Steuerung des Ladezustands und die Konfiguration des maximalen Stroms.


<DeviceConfig code={`chargers:

      token: # Tessie API Token
      location: tessiehome # Ort, Definieren Sie einen Ort, an dem das Tessie-Ladegerät funktioniert (always = immer, kein Geofence) oder ein benutzerdefinierter Ort (genauer Name), wie in Tessie definiert (Case sensitive)`}  />

## Templateparameter `tessie`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| Fahrzeugidentifikationsnummer<br/>`vin` | Wenn mehrere Fahrzeuge eines Herstellers vorhanden sind | Beispiel: `W...` | ja |
| Tessie API Token<br/>`token` |  |  | ja |
| Ort<br/>`location` | Definieren Sie einen Ort, an dem das Tessie-Ladegerät funktioniert (always = immer, kein Geofence) oder ein benutzerdefinierter Ort (genauer Name), wie in Tessie definiert (Case sensitive) | Beispiel: `tessiehome` | ja |
