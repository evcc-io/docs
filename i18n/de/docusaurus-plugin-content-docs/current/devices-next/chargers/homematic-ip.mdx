## Templateparameter `homematic`

| Name | Erklärung | Wert | Optional |
|-----------|-----------|------|----------|
| IP-Adresse oder Hostname<br/>`host` |  | Beispiel: `192.0.2.2` | ja |
| XML-RPC-Server Port-Nummer<br/>`port` |  | Standard: `2010`<br/>Beispiel: `BidCos-Wired=2000, BidCos-RF=2001, HmIP=2010` | nein |
| Geräteadresse/Seriennummer<br/>`device` | Homematic Geräte Id, wie im CCU Webfrontend angezeigt. | Beispiel: `0001EE89AAD848` | ja |
| Benutzerkonto<br/>`user` | bspw. E-Mail Adresse, User Id, etc. |  | nein |
| Passwort<br/>`password` | Bei führenden Nullen bitte in einfache Hochkommata setzen |  | nein |
| Standby-Leistung<br/>`standbypower` | Leistung oberhalb des angegebenen Wertes wird als Ladeleistung gewertet | Standard: `15`<br/>Einheit: W | nein |
| Kanalnummer des Power-Meters<br/>`meterchannel` | Kanalnummer des Messwertkanals, wie im CCU Webfrontend angezeigt. | Standard: `6`<br/>Beispiel: `HMIP-PSM=6, HMIP-FSM+HMIP-FSM16=5, HM=2` | ja |
| Kanalnummer der schaltbaren Steckdose<br/>`switchchannel` | Kanalnummer der schaltbaren Steckdose, wie im CCU Webfrontend angezeigt. | Standard: `3`<br/>Beispiel: `HMIP-PSM=3, HMIP-FSM+HMIP-FSM16=2, HM=1` | ja |
| XML-RPC API Cache Zeitraum<br/>`cache` | Bei Problemen mit dem Duty Cycle setze den Cache auf bspw 30s. | Standard: `1s`<br/>Beispiel: `5m` | nein |
| Integriertes Gerät<br/>`integrateddevice` | Fest angeschlossenes Gerät. Keine Ladevorgänge |  | nein |
| Wärmeerzeuger<br/>`heating` | Zeigt °C anstatt % an |  | nein |
| Icon<br/>`icon` | Wird in der Benutzeroberfläche angezeigt | Auswahl: `car`, `bike`, `bus`, `moped`, `motorcycle`<br/>, `rocket`, `scooter`, `taxi`, `tractor`, `rickshaw`<br/>, `shuttle`, `van`, `airpurifier`, `battery`, `bulb`<br/>, `climate`, `coffeemaker`, `compute`, `cooking`, `cooler`<br/>, `desktop`, `device`, `dishwasher`, `dryer`, `floorlamp`<br/>, `generic`, `heater`, `heatexchange`, `heatpump`, `kettle`<br/>, `laundry`, `laundry2`, `machine`, `meter`, `microwave`<br/>, `pump`, `smartconsumer`, `tool`, `waterheater` | nein |
