## Template Parameter `octopus-productcode`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Product Code<br/>`productCode` | The tariff code for your energy contract. Make sure this is set to your import tariff code. | Beispiel: `AGILE-FLEX-22-11-25` | yes |
| Region<br/>`region` | The DNO region you are located in. More information: https://www.energy-stats.uk/dno-region-codes-explained/ | Auswahl: `A`, `B`, `C`, `D`, `E`<br/>, `F`, `G`, `H`, `J`, `K`<br/>, `L`, `M`, `N`, `P` | yes |
| Direct debit tariff<br/>`directDebit` | Use Direct Debit tariff rates. | Standard: `true` | no |
