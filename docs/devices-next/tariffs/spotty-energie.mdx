## Template Parameter `spottyenergy`

| Name | Explanation | Value | Optional |

| Price type<br/>`pricetype` | Price type, either spotmarket price, consumption price or generation compensation (if contractually agreed), more info at https://www.spottyenergie.at/blog/energie-smart-produzieren | Standard: `CONSUMPTION`<br/>Auswahl: `MARKET`, `CONSUMPTION`, `GENERATION` | yes |
| Charge<br/>`charges` | Additional fixed charge per kWh (e.g. 0.05 for 5 cents) |  | no |
| Tax<br/>`tax` | Additional percentage charge (e.g. 0.2 for 20%) |  | no |
