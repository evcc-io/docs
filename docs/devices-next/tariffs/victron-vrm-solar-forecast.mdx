## Template Parameter `victron`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|

| API access token<br/>`token` | Tokens can be created in VRM in Preferences->Integrations |  | yes |
| Interval<br/>`interval` |  | Standard: `1h` | no |
