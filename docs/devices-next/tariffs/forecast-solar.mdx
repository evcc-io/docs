# Forecast.Solar

<DeviceFeatures features="Global" />

[forecast.solar](https://forecast.solar) can be used for free. Paid plans can also be used by specifying an API key.


<DeviceConfig code={`tariffs:
    solar:
      - type: template
        template: forecast-solar
        lat: 55.7351 # Latitude
        lon: 9.1275 # Longitude
        dec: 25 # Decline, 0 = horizontal, 90 = vertical
        kwp: 9.8 # Maximum generator power (kWp)
        az: 180 # Azimuth, -180 = north, -90 = east, 0 = south, 90 = west, 180 = north`} advanced={`tariffs:
    solar:
      - type: template

        horizon: 0,0,15,30,45,60,60,60,45,30,15,0 # Horizon, Simulates terrain shadows, [more information](https://doc.forecast.solar/horizon) (optional)

## Template Parameter `forecast-solar`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Latitude<br/>`lat` |  | Beispiel: `55.7351` | yes |
| Longitude<br/>`lon` |  | Beispiel: `9.1275` | yes |
| Decline<br/>`dec` | 0 = horizontal, 90 = vertical | Beispiel: `25` | yes |
| Maximum generator power<br/>`kwp` |  | Beispiel: `9.8`<br/>Einheit: kWp | yes |
| Azimuth<br/>`az` | -180 = north, -90 = east, 0 = south, 90 = west, 180 = north | Beispiel: `180` | yes |
| Horizon<br/>`horizon` | Simulates terrain shadows, \[more information\](https://doc.forecast.solar/horizon) | Beispiel: `0,0,15,30,45,60,60,60,45,30,15,0` | no |
| API key<br/>`apikey` |  |  | no |
| `interval` |  | Standard: `1h` | no |
