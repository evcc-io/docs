# Fraunhofer ISE

<DeviceFeatures features="EU" />

Day-ahead forecast of energy prices (per kWh) on the exchange. No prior registration for https://api.energy-charts.info/ necessary. Can be used for dynamic electricity tariffs, for example, where the supplier does not yet offer a price forecast on the customer interface.

      charges: # Charge, Additional fixed charge per kWh (e.g. 0.05 for 5 cents) (optional)

## Template Parameter `energy-charts-api`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| `bzn` | Bidding zones - https://api.energy-charts.info/#/prices/day_ahead_price_price_get | Standard: `DE-LU`<br/>Auswahl: `AT`, `BE`, `CH`, `CZ`, `DE-LU`<br/>, `DE-AT-LU`, `DK1`, `DK2`, `FR`, `HU`<br/>, `IT-NORTH`, `NL`, `NO2`, `PL`, `SE4`<br/>, `SI` | yes |
| Charge<br/>`charges` | Additional fixed charge per kWh (e.g. 0.05 for 5 cents) |  | no |
| Tax<br/>`tax` | Additional percentage charge (e.g. 0.2 for 20%) |  | no |
| Formula<br/>`formula` | Individual formula for calculating the price | Beispiel: `math.Max((price + charges) * (1 + tax), 0.0)` | no |
