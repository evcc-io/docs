## Template Parameter `easee`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Username<br/>`user` | Email address |  | yes |
| Password<br/>`password` | same as Easee app or the web portal (https://easee.cloud) |  | yes |
| `charger` |  | Beispiel: `EH______` | yes |
| Timeout<br/>`timeout` | Timeout specifically for Easee API interactions. Can reduce warnings and errors in case of lagging Easee API. | Standard: `20s`<br/>Beispiel: `10s` | no |
| `authorize` | Controls wether evcc shall perform authentication against charger. Benefit is a contolled start of charging. Not compatible with RFID identification of vehicles. |  | no |
