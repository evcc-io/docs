## Template Parameter `luxtronik`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| `modbus` |  | Auswahl: `tcpip` | no |
| `heatoffset` | Temperature boost offset for heating. | Standard: `0.0`<br/>Beispiel: `2.0`<br/>Einheit: K | no |
| `wwoffset` | Temperature boost offset for hot water. Possibly increases wear on compressor. | Standard: `0.0`<br/>Beispiel: `8.5`<br/>Einheit: K | no |
