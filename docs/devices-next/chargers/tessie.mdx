## Template Parameter `tessie`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Vehicle Identification Number<br/>`vin` | If you own multiple vehicles from the same manufacturer | Beispiel: `W...` | yes |
| Tessie API Token<br/>`token` |  |  | yes |
| Location<br/>`location` | Define a location where the Tessie charger will work (always = always, no geofence) or a custom location (exact name) as defined in Tessie (Case sensitive) | Beispiel: `tessiehome` | yes |
