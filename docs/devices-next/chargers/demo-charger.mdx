## Template Parameter `demo-charger`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Charge status<br/>`status` |  | Standard: `A`<br/>Auswahl: `A`, `B`, `C` | yes |
| Power<br/>`power` |  |  | no |
| Enabled<br/>`enabled` |  | Standard: `true` | no |
| Maximum amperage<br/>`maxcurrent` | The maximum amperage per connected phase with which the car shuold be charged | Beispiel: `16`<br/>Einheit: A | no |
| Phase switching<br/>`phases1p3p` |  |  | no |
