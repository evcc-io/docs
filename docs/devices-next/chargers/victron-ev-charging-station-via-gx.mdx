# Victron EV Charging Station (via GX)

<DeviceFeatures features="" />

Enter the host of the GX device (not the charger). The charger has to be in manual mode and Modbus has to be configured for ID 100.


<DeviceConfig code={`chargers:
    - name: my_charger
      type: template
      template: victron
      
      # Modbus TCP
      modbus: tcpip
      id: 100
      host: 192.0.2.2 # Hostname
      port: 502 # Port`}  />

## Template Parameter `victron`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| `modbus` |  | Auswahl: `tcpip` | no |
