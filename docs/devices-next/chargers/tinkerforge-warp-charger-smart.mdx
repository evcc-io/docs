## Template Parameter `tinkerforge-warp`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|

| Port<br/>`port` | MQTT broker port | Standard: `1883` | no |
| Topic<br/>`topic` | Topic (omit leading /) | Standard: `warp` | no |
| Timeout<br/>`timeout` | Don't accept values older than this value | Standard: `30s`<br/>Beispiel: `10s` | no |
