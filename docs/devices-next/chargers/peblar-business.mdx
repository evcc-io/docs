# Peblar Business

<DeviceFeatures features="1p3p,mA,sponsorship" />

Peblar chargers with firmware version 1.6 and onwards support Modbus TCP via port 502. The Modbus server needs to be enabled via the charger web interface. Ensure that smart charging strategies are disabled and set to Default.


<DeviceConfig code={`chargers:
    - name: my_charger
      type: template
      template: peblar
      
      # Modbus TCP
      modbus: tcpip
      id: 1
      host: 192.0.2.2 # Hostname
      port: 502 # Port`}  />

## Template Parameter `peblar`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| `modbus` |  | Auswahl: `tcpip` | no |
