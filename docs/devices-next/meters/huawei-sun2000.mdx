## Template Parameter `huawei-sun2000`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Modbus Type<br/>`modbus` |  | Auswahl: `rs485`, `tcpip` | no |
| Timeout<br/>`timeout` |  | Standard: `15s`<br/>Beispiel: `10s` | no |
| Battery storage unit index<br/>`storageunit` |  | Standard: `1` | no |
| Maximum charge power<br/>`maxchargepower` | For forced charging of the battery. | Standard: `10000`<br/>Einheit: W | no |
| Battery capacity<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | no |
| Maximum AC power of the hybrid inverter<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | no |
| Inverter cascade<br/>`forceaccharging` | Keep AC charging active to charge the storage from other inverters via AC. Prevents stand-by. |  | no |
