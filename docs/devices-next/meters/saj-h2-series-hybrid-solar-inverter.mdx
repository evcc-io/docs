## Template Parameter `saj-h2`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| `modbus` |  | Auswahl: `rs485`, `tcpip` | no |
| Battery capacity<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | no |
| Default mode for battery control<br/>`defaultmode` | Inverter falls back to this mode after charging the battery or after stopping discharge. | Standard: `2` | no |
| Minimum charge<br/>`minsoc` | Lower limit when discharging the battery in normal operation | Standard: `20`<br/>Beispiel: `25`<br/>Einheit: % | no |
| Maximum charge<br/>`maxsoc` | Upper limit when charging the battery from the grid | Standard: `95`<br/>Beispiel: `95`<br/>Einheit: % | no |
