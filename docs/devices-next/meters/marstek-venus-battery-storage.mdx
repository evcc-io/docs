## Template Parameter `marstek-venus`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Modbus Type<br/>`modbus` |  | Auswahl: `rs485` | no |
| Battery capacity<br/>`capacity` | Venus-E 5.12 kWh, Venus-C 2.56 kWh | Standard: `5.12`<br/>Beispiel: `50`<br/>Einheit: kWh | no |
| Minimum charge<br/>`minsoc` | Venus can only be discharged down to 11%. | Standard: `11`<br/>Beispiel: `11`<br/>Einheit: % | no |
| Maximum charge<br/>`maxsoc` | Upper limit when charging the battery from the grid | Standard: `100`<br/>Beispiel: `95`<br/>Einheit: % | no |
| Maximum charge power<br/>`maxchargepower` | 2500W is the maximum for Venus. | Standard: `2500`<br/>Einheit: W | no |
| Work mode for Normal state<br/>`work_mode_normal` | 0=manual, 1=anti-feed, 2=trade mode. | Standard: `1` | no |
