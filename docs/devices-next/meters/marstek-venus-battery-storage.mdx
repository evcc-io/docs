## Template Parameter `marstek-venus`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| `modbus` |  | Auswahl: `rs485` | no |
| Battery capacity<br/>`capacity` | Venus-E 5.12 kWh, Venus-C 2.56 kWh | Standard: `5.12`<br/>Beispiel: `50`<br/>Einheit: kWh | no |
| Venus can only be discharged down to 11%.<br/>`minsoc` | Lower limit when discharging the battery in normal operation | Standard: `11`<br/>Beispiel: `25`<br/>Einheit: % | no |
| Maximum charge<br/>`maxsoc` | Upper limit when charging the battery from the grid | Standard: `100`<br/>Beispiel: `95`<br/>Einheit: % | no |
| Charge power in watts for Charge mode. 2500W is the maximum for Venus.<br/>`maxchargepower` | For forced charging of the battery. | Standard: `2500`<br/>Einheit: W | no |
| Work mode for Normal state. 0 manual, 1 anti-feed, 2 trade mode.<br/>`work_mode_normal` |  | Standard: `1` | no |
