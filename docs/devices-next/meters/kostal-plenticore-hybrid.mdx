# Kostal Plenticore Hybrid

<div className="device-detail-container">

Only a single system may access the inverter! For active battery control the function external battery control via Modbus must be activated using installer access.
**Grid charging is not available!** _see also https://github.com/evcc-io/evcc/wiki/Kostal-Plenticore_

## Template Parameter `kostal-plenticore`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Modbus Type<br/>`modbus` |  | Auswahl: `tcpip` | no |
| Endianness (Little/Big)<br/>`endianness` |  | Standard: `little`<br/>Auswahl: `little`, `big` | no |
| Battery capacity<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | no |
| Maximum AC power of the hybrid inverter<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | no |
| Minimum charge<br/>`minsoc` | Lower limit when discharging the battery in normal operation | Beispiel: `25`<br/>Einheit: % | no |
| Maximum charge<br/>`maxsoc` | Upper limit when charging the battery from the grid | Beispiel: `95`<br/>Einheit: % | no |
| Watchdog<br/>`watchdog` |  | Standard: `60s` | no |
