## Template Parameter `rct-power`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| IP address or hostname<br/>`host` |  | Beispiel: `192.0.2.2` | yes |
| Battery capacity<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | no |
| Minimum charge<br/>`minsoc` | Lower limit when discharging the battery in normal operation | Standard: `7`<br/>Beispiel: `25`<br/>Einheit: % | no |
| Maximum charge<br/>`maxsoc` | Upper limit when charging the battery from the grid | Standard: `97`<br/>Beispiel: `95`<br/>Einheit: % | no |
| Cache<br/>`cache` | Time interval with when data should be reloaded from the vehicle | Standard: `30s`<br/>Beispiel: `5m` | no |
| Maximum charge power<br/>`maxchargepower` | For forced charging of the battery. | Einheit: W | no |
| External power<br/>`externalpower` | Query external power of all devices connected to S0 |  | no |
