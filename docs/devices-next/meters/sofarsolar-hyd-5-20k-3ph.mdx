# SofarSolar HYD 5…20K-3PH

<div className="device-detail-container">
  <div className="device-detail-content">

LSE-3 logger stick using a LAN connection and ModBus TCP via the port 8899 is the easiest connection. The LSW-3 WiFi stick is not supported. For a RS485 serial connection using the inverter's COM port the inverter's side must be properly terminated.

## Template Parameter `sofarsolar-g3`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Modbus Type<br/>`modbus` |  | Auswahl: `tcpip`, `rs485` | no |
| Battery storage unit index<br/>`storageunit` | In case of a BTS storage not the address of a BTS 5K battery module, but the storage tower (BTS 5K-BDU control unit with 1-4 BTS 5K modules). | Standard: `1` | no |
| Battery capacity<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | no |
| Maximum AC power of the hybrid inverter<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | no |
| Default mode for battery control<br/>`defaultmode` | Valid values are 0 (self use), 1 (time of use), 2 (timing mode), 4 (peak-shaving mode) |  | no |
