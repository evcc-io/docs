## Template Parameter `sma-hybrid`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| Modbus Type<br/>`modbus` | ModbusTCP server needs to be enabled. | Auswahl: `tcpip` | no |
| Battery capacity<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | no |
| Maximum AC power of the hybrid inverter<br/>`maxacpower` |  | Beispiel: `5000`<br/>Einheit: W | no |
| Maximum charge power<br/>`maxchargepower` | For forced charging of the battery. | Standard: `4200`<br/>Einheit: W | no |
| Watchdog<br/>`watchdog` |  | Standard: `60s` | no |
