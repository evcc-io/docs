## Template Parameter `sunspec-inverter-control`

| Name | Explanation | Value | Optional |
|-----------|-------------|-------|----------|
| `modbus` |  | Auswahl: `tcpip`, `rs485` | no |
| Maximum charge power<br/>`maxchargerate` | For forced charging of the battery in percent in relation to the maximum charge power of the battery inverter. | Standard: `100`<br/>Einheit: % | no |
| Battery capacity<br/>`capacity` |  | Beispiel: `50`<br/>Einheit: kWh | no |
